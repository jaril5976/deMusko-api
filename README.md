# Project Name : DeMusko-api

## How to setup project for development

1 Clone repo.

2 Chekout to `dev` branch.
```
git fetch --all
git checkout dev
git pull origin dev
```

3 Install dependencies. 
```
npm install
```

4 Run DeMusko. 
```
npm start
```

5 Run API-DOCS. 
```
npm run api-docs
```

### your api server is localhost:8080
